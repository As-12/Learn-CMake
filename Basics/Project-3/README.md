# Project3 - Static library 

A simple project with a compiled static library.

```
Project-3$ tree
.
├── CMakeLists.txt
├── Math
    └── sum.hpp
│   └── src
│       └── sum.cpp
└── src
    └── main.cpp
```

## To compile:

mkdir Build
cd Build
cmake ..
make

## To Run
The executable will generate on Build folder. 
./Project-3
