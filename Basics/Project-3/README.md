# Project2 - Multiple source files CMake. 

A simple self-contained project with separated header and source files

```
Project-2$ tree
.
├── CMakeLists.txt
├── include
│   └── sum.hpp
└── src
    ├── sum.cpp
    └── main.cpp
```

## To compile:

mkdir Build
cd Build
cmake ..
make

## To Run
The executable will generate on Build folder. 
./Project-2
